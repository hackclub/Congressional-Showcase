The Platform

How It Works

SafeZone is a progressive web app built for natural disaster survival. It pulls real-time alert data from FEMA and ArcGIS services to notify users of active threats including hurricanes, floods, wildfires, and severe storms. Service Workers enable offline functionality, ensuring the app remains usable even when network connectivity is lost.

The safe spot locator uses Google Maps API combined with FEMA shelter data to identify and route users to the nearest available shelters. Emergency communication features help users coordinate with contacts and access critical resources during crises. The entire platform runs in the browser with no installation required, making it instantly accessible on any device.
