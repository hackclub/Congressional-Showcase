The App

How It Works

Motio is a mobile iOS app that helps users improve their driving by tracking data from the phone's GPS and accelerometer. The app provides detailed analysis of driving errors along with suggestions for improvement, making every trip a learning opportunity.

Beyond individual improvement, Motio includes social features that let users add friends, compete on driving scores, and view each other's sessions. The app is built with a React Native frontend, a FastAPI backend, and Firebase for authentication and database management.
