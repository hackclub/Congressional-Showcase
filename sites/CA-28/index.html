The App

About Fire Up

Existing wildfire readiness initiatives lack real-time prediction and detection accessible to both first responders and the general public. Fire Up is the first solution to bring together dynamic fire prediction, live fire and smoke detection, evacuation route planning, and educational guidance in one integrated experience.

The vision for Fire Up is not just to report danger, but to help people react intelligently and quickly. In the future, the platform is designed with the potential to work in conjunction with fire drones, connecting software intelligence with hardware-based wildfire response systems.

Main Features

  • Dynamic wildfire prediction mapping
  • Integration of low-cost Raspberry Pi set-up for fire/smoke detection
  • Evacuation planning and safer route support
  • Preparedness information and built-in SOS functionality

The Developer

About the Developer: Anika Jha

Hi, I’m Anika Jha. I'm a sophomore from Arcadia High School, and I'm passionate about creating technology that can make a real impact. I love combining coding, engineering, design, and scientific problem-solving to build projects that are both meaningful and practical.

I created Fire Up to address a problem that affects millions of people: wildfire risk. Living in California, I have seen how important early awareness, fast response, and clear safety planning can be. My family and I were almost evacuated during the Palisades & Eaton fires, and I wanted to build something that supports both information and action in a unique way.

Prediction Map

Prediction Map

• Uses live inputs from Open-Meteo API, NASA AppEEARS, and CalFire GIS data.
• Processes data on land cover, vegetation index (NDVI), soil moisture, and meteorological variables.
• Employs a custom LightGBM machine-learning model to estimate:
    ° Wildfire likelihood
    ° Acres potentially affected
    ° Fire duration / severity
• Map visualization built with Plotly Mapbox - counties shaded by risk, marker size scales with predicted duration, and radius scales with acres affected.

Detection

Detection

• Integrates with Raspberry Pi 5 + sensor modules (temperature, humidity, light, camera).
• Runs a TensorFlow Keras CNN trained to 95 % accuracy for detecting fire / smoke / safe frames.
• Users can paste their Flask API URL directly into the app to stream live detection results from their own devices.

Detection Tech Stack

Detection Tech Stack

The Raspberry Pi module allows you to stream real-time environmental and camera data directly into the Fire Up app.
It runs a local Flask server that the web app connects to in order to display live sensor data and video.

This script captures:

• Temperature & Humidity from a DHT20 sensor
• Light intensity (lux) from a BH1750 sensor
• Live video feed from the Raspberry Pi camera
• And exposes them via a Flask API that connects to Fire Up’s Detections page
• The live camera feed is fed into a custom TensorFlow Keras CNN of 95% accuracy that classifies whether there is fire, smoke, or neither
