The Application

How It Works

Isomer Enumerator uses recursive backtracking and object-oriented programming to generate all valid molecular variants for chemistry education. The application takes a molecular formula as input and systematically explores every possible structural arrangement that satisfies chemical bonding rules.

The backend is built with Python and connected to a JavaScript frontend via FastAPI. Graph and node structures represent chemical objects, allowing the system to model atoms as nodes and bonds as edges. This approach enables efficient traversal and validation of molecular configurations while enforcing electron counts and formal charge constraints.
