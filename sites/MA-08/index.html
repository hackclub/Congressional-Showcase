The Game

How It Works

Inspect features two core game modes. In "True or False," players face three levels of AI-generated headlines and must determine whether each one is real or fabricated. In "Which One," players are shown pairs of AI-generated video-text combinations and must identify the more extreme version. Instant feedback is provided after every answer.

The challenges are created using GPT-4.1, GPT-5, and Sora 2. Videos are generated with Sora 2 to create realistic but synthetic content. Anti-cheat measures prevent copying challenge text, and watermarks are embedded throughout. The app is built with React and CSS on the frontend, with Firebase handling the database, storage, and deployment.
