The Platform

How It Works

Verdis combines drone-captured aerial imagery with advanced machine learning to monitor crop health at scale. Drones survey farmland and capture high-resolution images, which are processed through a SIFT (Scale-Invariant Feature Transform) algorithm using OpenCV to stitch together comprehensive field-wide health maps.

Individual crop images are analyzed by a PyTorch convolutional neural network trained to identify specific disease strains with high accuracy. The detection results feed into a large language model that acts as an AI advisor, interpreting the technical diagnoses and generating plain-language treatment recommendations tailored to each farmer's situation.

The frontend is built with React and TypeScript for a responsive, real-time dashboard experience, while the backend runs on FastAPI with Python, handling image processing, ML inference, and LLM queries. Together, these components create an end-to-end precision agriculture platform accessible from any device.
