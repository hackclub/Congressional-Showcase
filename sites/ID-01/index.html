Planelooly – A game about connecting airports.

Planelooly is an online game about designing routes for a passenger airline. The game helps to teach players about American geography and the airline industry, with the game featuring over 500 US airports. The game also hopes to excite players about fields including air traffic control.

Planelooly was built using Svelte, SvelteKit, TypeScript, and TailwindCSS.
