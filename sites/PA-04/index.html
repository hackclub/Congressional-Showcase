The App

What InfiniteMonkey Does

InfiniteMonkey addresses resource consumption concerns with large language models by using lightweight Markov chain analysis. The application scrapes web text to build a corpus, analyzes word and phrase relationships based on configurable n-gram sizes, and generates new text using probability modeling.

Output coherence is enhanced through synonym replacement and part-of-speech tagging. Users interact through a graphical interface where they input questions and the program retrieves relevant online content for analysis.

Named after the Infinite Monkey Theorem, the program uses probability-based "guessing" for word succession.
